Padding, Display and Margins

https://getbootstrap.com/docs/4.1/utilities/spacing/

The $spacer variable is the same size as the font size. We multiply it to get spacing for margin and padding.

  • 0 = 0 * $spacer
  • 1 = 0.25 * $spacer
  • 2 = 0.5 * $spacer
  • 3 = 1 * $spacer
  • 4 = 1.5 * $spacer
  • 5 = 3 * $spacer
