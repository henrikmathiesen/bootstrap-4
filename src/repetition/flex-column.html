Flex Column

https://getbootstrap.com/docs/4.1/utilities/flex/

  • flex-column (vertical direction) is not default and needs to be set with css class
  • can align all items on y-axis (justify-content-x)
  • can not align self on y-axis (use margin auto instead)
  • can align all items on x-axis (align-items-x)
  • can align self on x-axis (align-self-x)
